Difference between var, let and const in Js.

Var:

With var we declare a identity.If declared without intitialisation they hold value as undefined.Scope, the region of accecibility for a paricular identifier. There are two type of scope global scope and function scope.

Global scope:
If a identifier is declared with a global scope it can be accessed from any part of the program.

Function scope:
If a identifier is declared with in a function by using var then it won't be accessable out of the function.

Let:

Similar to var let is also used to declare an identifier,but let is sccoped differently than var.Let is blocked scoped or it's bounded in the block of in{} as compared to function scoped var and we get RefrenceError when acessing a variable declared by let.

Const:

Const is almost exactly the same as let. However, we cannot change the value of const variable once defined
